# UI sound effects

Menu navigation samples from **Final Fantasy VI** (SNES), used under
permission from Square Enix.

| File | Game action | Source pack name |
|------|-------------|------------------|
| `cursor.wav` | Move highlight | `DCMenuTing.wav` |
| `confirm.wav` | Confirm / select | `1BItemMenuItng.wav` |
| `cancel.wav` | Cancel / close | `78Closing.wav` |
| `save.wav` | Save success | `5ASaveTing.wav` |
| `buy-sell.wav` | Shop buy/sell | `BFBuyingSelling.wav` |
| `cure-menu.wav` | Menu heal / cure | `EECureInMenu.wav` |

Do not redistribute these files outside this project without the same clearance.
