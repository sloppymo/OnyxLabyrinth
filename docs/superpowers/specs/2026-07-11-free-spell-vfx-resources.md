# Free Spell VFX Sprite Resources — Curated List for OnyxLabyrinth

This is a curated list of free (or mostly free) pixel-art spell/VFX sprite resources found on the web. The game needs 16-bit style horizontal PNG strip sheets for fire, cold, lightning, poison, holy, heal, buff, status, and summon effects. Each entry notes the license, contents, and how it might fit the project's spell list.

---

## Legend

- **CC0** — Public domain, no attribution required, commercial use OK.
- **CC-BY** — Requires attribution in credits/readme.
- **Royalty-free** — Free to use in commercial projects, usually no attribution, but do not resell the raw assets.
- **Name-your-own-price** — Often downloadable for $0 on itch.io if the author allows it.
- **Paid** — Not free, but listed because it is high-quality and cheap; skip if you need zero-cost assets only.

## Top Recommendations

### 1. Pixel Art Spells by DevWizard (OpenGameArt)

- **URL:** https://opengameart.org/content/pixel-art-spells
- **License:** CC0
- **Cost:** Free
- **Style:** 16-bit pixel art, 16×16 base
- **Contents:** 23+ spell projectiles: arcane bolt, fireball, firebomb, ice lance, light bolt, magic orb, magic ray, magic shield, plant missile, rock sling, water bolt, wind bolt, darkness bolt, etc.
- **Why it fits:** True CC0, explicitly 16-bit, and covers fire, ice, light, arcane, and earth projectiles. Perfect as starting material for Ember, Fire Bolt, Frostbite, Guiding Bolt, and Arcane Ward projectiles.
- **Caveat:** Mostly projectiles, not bursts/fields. You may need to build impact bursts from other packs or draw them.

## Workflow tips for using these assets

1. **Download the PNG/spritesheet versions**, not GIFs if you can avoid them. GIFs have matte/fringing issues.
2. **Check dimensions and frame layout.** Many packs use 2D grids (e.g., 6×2) rather than a single horizontal row. You may need to slice/reassemble frames into a single horizontal strip for the engine.
3. **Recolor to match the game's palette.** The game uses specific colors per element (fire = `#ff8c42`, cold = `#80e0ff`, lightning = `#ffd769`, poison = `#c080ff`, divine = `#ffe8a0`, heal = `#6fe06f`). Shifting hues in Aseprite or GIMP is quick.
4. **Remove or replace background.** All effects must have transparent backgrounds. Some packs include black backgrounds by default; use the alpha/transparency versions.
5. **Resize carefully.** If an asset is too large, resize with nearest-neighbor interpolation to keep crisp pixels.
6. **Attribute CC-BY assets** in the project's credits or `ATTRIBUTION.md`.

## Recommendation summary

If you want **zero-cost, no-attribution** assets: start with **Pixel Art Spells (CC0)**, **Foozle Pixel Magic Effects (CC0)**, and **Willibab Simple Spell Effects**.

If you want **more variety and are OK with royalty-free (not CC0)** packs: add **pewas Pixel RPG VFX Pack**, **ansimuz Free Magic Pack 9**, and **Frostwindz Portal/Priest**.

If you are willing to **spend a small amount** for high-quality status and elemental effects: **finalbossblues Pixel Animations and Effects** ($6) and **Marco Brito Pixel Magic Effects Pack** ($3) are excellent.
